``ssc-pkg``
============

Utility to build and package simfiles for easy distribution

Installation
------------

Run (the standard) ``pip3 install -e .`` as in `.travis.yml`_.

Prerequisites
^^^^^^^^^^^^^

See `setup.py`_ for exact Python requirements,
or `.travis.yml`_ for other requirements.
